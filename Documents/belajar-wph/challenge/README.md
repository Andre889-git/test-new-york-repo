# Challenge 1 - Simple Landing Page with Html and CSS

## Description

In this assignment, you will implement the basic structure and visual style of a simple webpage using HTML and Vanilla CSS (without any additional CSS frameworks or libraries).

The goal of this assignment is to test your understanding of basic HTML structure (Semantic and non-semantic), the use of CSS selectors, the box model, and your ability to transform your code into design. And also it will be a good practice to use git as our versioning tools.

## Folder Structure

Below is the initial folder structure that has been provided for your project:

- `Css/style.css`: This is where you will write all the CSS code to style your webpage.
- `index.html`: This is the main HTML file where you will create the structure of your webpage.

## Assignment Objectives

1.  **Semantic HTML Structure:** Create a good and semantic HTML structure according to the layout in the Figma design. Use the appropriate HTML tags for each element (e.g., `<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`, `<h1>` to `<h6>`, `<p>`, `<ul>`, `<li>`, `<a>`, `<img>`, etc.).
2.  **Visual Styling with Vanilla CSS:** Implement the visual style of the webpage using pure CSS. Pay attention to things like:
    - Element layout (e.g., using Flexbox or CSS Grid if necessary).
    - Colors, fonts, text sizes, and other visual properties.
    - Spacing (margin and padding) between elements.
3. **Able to commit and push to GitHub with at least 3 commits.**


## Example Workflow

1.  **Clone the project** Open the provided GitHub URL and clone the project to your local machine to begin working on it..
2.  **HTML Structure:** Open the `index.html` file and write the basic HTML structure according to the design. Identify the main sections of the page (header, navigation, main content, footer, etc.) and use relevant HTML tags.
3.  **CSS Styling:** Open and edit the `Css/style.css` file to provide visual styles to your HTML elements. Use CSS selectors to target the appropriate elements and apply CSS properties according to the design.
4.  **Git Commit:** Commit every action you make with a minimum of 3 commits, and push them to the provided GitHub repository.

## Evaluation System

The evaluation for this assignment will be based on the following criteria:

1.  **Semantic HTML Usage:** The correct and appropriate use of HTML tags according to the content structure.
2.  **Effective CSS Usage:** The correct and efficient use of CSS selectors, as well as understanding of basic CSS properties.
3.  **Code Readability and Organization:** HTML and CSS code that is well-structured, easy to read, and commented where necessary.
4. **Git Management:** Able to use basic Git workflows such as git add, git commit, and git push.
